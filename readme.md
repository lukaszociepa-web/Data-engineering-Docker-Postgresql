# Data Engineering - Docker & PostgreSQL

## Tech Stack

* Python
* PostgreSQL
* Docker
* pandas
* SQLAlchemy

## Goal

The goal of this project is to:

* download data from external sources,
* process and filter the data,
* load the data into PostgreSQL,
* build a simple ETL pipeline.

## Environment

* Ubuntu WSL
* VS Code
* Docker Desktop
